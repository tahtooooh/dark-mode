this python script is for changing your pdf files to dark mode, you need to put the book in the same file as the script and run the script, it will downlaod the dark mode version automatically 
to run it do this command :
python pdf_darkmode.py --input example.pdf

ps: you can change the size of the new file if you change the dpi


try not to use special characters in the name if the book; and don't use spaces.
